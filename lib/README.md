# FoodPay — Setup Guide

## Project Structure
```
lib/
├── main.dart                    ← App entry, colors, Cart, FoodItem models
├── firebase_options.dart        ← Generated by FlutterFire CLI
└── screens/
    ├── splash_screen.dart       ← Animated splash, auto-routes based on auth
    ├── menu_screen.dart         ← Food menu with categories + cart badge
    ├── cart_screen.dart         ← Cart with qty control + order summary
    ├── payment_screen.dart      ← Hand gesture payment (ML Kit Pose)
    ├── success_screen.dart      ← Animated success + Firestore order saved
    └── auth/
        ├── login_screen.dart    ← Email/password + Face login button
        ├── register_screen.dart ← Register + optional face enroll
        └── face_auth_screen.dart ← Camera + ML Kit face liveness
```

## Step 1 — Firebase Setup
1. Go to https://console.firebase.google.com
2. Create project (e.g. "foodpay")
3. Add Android app → package name from your `android/app/build.gradle`
4. Download `google-services.json` → place in `android/app/`
5. Enable **Email/Password** auth in Authentication → Sign-in methods
6. Enable **Firestore** in Firestore Database (start in test mode first)
7. Paste `firestore_rules.txt` content into Firestore → Rules → Publish

## Step 2 — FlutterFire CLI
```bash
dart pub global activate flutterfire_cli
flutterfire configure
```
This generates `lib/firebase_options.dart` automatically.

## Step 3 — Android Permissions
Add to `android/app/src/main/AndroidManifest.xml` inside `<manifest>`:
```xml
<uses-permission android:name="android.permission.CAMERA"/>
<uses-permission android:name="android.permission.INTERNET"/>
```

Add inside `<application>`:
```xml
<meta-data
    android:name="com.google.mlkit.vision.DEPENDENCIES"
    android:value="face,pose" />
```

## Step 4 — android/app/build.gradle
```groovy
android {
    defaultConfig {
        minSdkVersion 21   // ML Kit requires 21+
    }
}
```

## Step 5 — Run
```bash
flutter pub get
flutter run
```

## Feature Notes

### Face Liveness Detection
- Uses **ML Kit Face Detection** with `enableClassification: true`
- Detects `eyeOpenProbability` to verify blink (proves real person)
- Steps: Look straight → Blink once → Confirmed
- Falls back to password if camera fails or user taps "Use password instead"
- On Register: prompts to enroll face → saves `faceEnrolled: true` to Firestore

### Hand Gesture Payment
- Uses **ML Kit Pose Detection** (base model, streaming mode)
- Landmarks: `leftWrist`, `leftThumb`, `leftIndex`, `leftPinky`
- 👍 **Thumbs Up** = thumb Y significantly above other fingers → Confirm
- ✊ **Fist** = all fingertips close to wrist level → Cancel
- Must hold gesture for **1 second** to trigger (prevents accidental)
- Demo buttons at bottom for testing without real gesture
- On confirm: saves full order to Firestore `orders` collection, clears cart

### Firestore Collections
```
users/{uid}
  name: string
  email: string
  faceEnrolled: bool
  createdAt: timestamp

orders/{orderId}
  uid: string
  items: [{name, emoji, qty, price}]
  subtotal: number
  tax: number
  total: number
  paymentMethod: "gesture"
  status: "paid"
  createdAt: timestamp
```
